Saifi Furniture is a fully interactive, beautifully designed **React-based furniture showroom website** featuring smooth navigation, image lightbox galleries, category filters, animations, and modern UI/UX elements. The project demonstrates advanced React component architecture, reusable modules, and interactive features like a custom lightbox, dynamic product filters, and sections that scroll smoothly into view.

Tech Stack

- React 18
- Vite (fast bundler)
- Tailwind CSS for styling
- Lucide Icons for UI icons
- JavaScript (ES6+)
- CSS Animations + Transitions


-Installation & Setup

-Clone the repo
git clone https://github.com/your-username/saifi-furniture.git
cd saifi-furniture

-Install dependencies
npm install

-Start the development server
npm run dev
